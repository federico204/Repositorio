# AI Feature Prioritizer 🤖

Este proyecto automatiza la gestión de productos mediante el uso de IA Generativa. Transforma quejas de usuarios en decisiones estratégicas.

### 📋 Requisitos
- Python 3.10+
- Una API Key de Google Gemini (AI Studio).

### 🚀 Instalación y Uso
1. Clonar el repositorio.
2. Instalar dependencias: `pip install google-generativeai`.
3. Configurar tu clave: `genai.configure(api_key="TU_KEY")`.
4. Ejecutar el script: `python main.py`.

### 📂 Archivos Generados
- `backlog_priorizado.csv`: Reporte listo para importar en Excel o Jira.
